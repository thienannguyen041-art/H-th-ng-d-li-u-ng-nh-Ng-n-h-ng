Finance Analytics Portfolio Project

Offline Banking Sector Analytics Dashboard

An offline HTML-based dashboard for Vietnam banking sector analysis, designed to transform structured Excel data into sector KPIs, bank-level analysis, peer comparison, financial statement views, and customizable charts.

Sector overviewTrack banking KPIs and industry-level performance.
Bank analysisReview individual bank metrics, trends, and financial statements.
Peer comparisonCompare banks across profitability, asset quality, efficiency, and growth.
Chart builderCreate custom visualizations for research and reporting.
View GitHub Repository Read Project Notes
The full offline build is a large single HTML file because it embeds JavaScript libraries and default banking data. For CV use, this repository serves as the public portfolio page, while the full offline demo can be stored in Google Drive or uploaded manually as a release asset.
