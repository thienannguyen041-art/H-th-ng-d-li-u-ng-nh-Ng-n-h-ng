# Offline Banking Sector Analytics Dashboard

An offline HTML-based analytics dashboard for Vietnam banking sector analysis. The project is designed to transform structured Excel banking data into interactive sector KPIs, bank-level analysis, peer comparison, financial statement views, and customizable charts for financial decision support.

## Project overview

This project was built as a finance analytics tool rather than a static report. It is intended for users who want to update banking data locally from a structured Excel file and immediately review key banking indicators through an interactive dashboard.

## Key features

- Offline HTML dashboard for Vietnam banking sector analysis
- Excel-based data update workflow
- Sector overview module
- Individual bank analysis module
- Peer comparison module
- Financial statement review module
- Custom chart builder
- Methodology documentation for KPI interpretation
- Interactive charts for financial analysis and reporting

## Use cases

- Banking sector monitoring
- Equity research support
- Peer comparison and KPI tracking
- Financial statement review
- Research dashboard demonstration
- Finance/data analytics portfolio project

## Note

The full offline build contains embedded JavaScript libraries and default banking data, so the file is relatively large. For portfolio use, the recommended setup is:

1. Keep the full offline HTML file in Google Drive for download/demo.
2. Use this GitHub repository as the public portfolio page and source description.
3. Optionally publish a lighter GitHub Pages version that loads charting libraries from CDN.
